# IDP pressure reconstruction from PIV

This repository contains the minimal MATLAB implementation and supporting datasets associated with the article:

Dontu, A. M., Daliri, A., and Scarano, F.,
"Iterative domain partitioning in pressure from PIV"

## Contents
- `code/` : single-file MATLAB implementation corresponding to Appendix B
- `data/vc7/` : sample datasets used for representative reconstruction cases
- `figures/matlab_fig/` : MATLAB figure files corresponding to article results
- `figures/exported/` : exported images for quick viewing

## Usage
Open `code/idp_appendixB_example.m` in MATLAB and adjust file paths to the desired dataset in `data/vc7/`.

## Notes
The code is intended as a minimal reproducibility example accompanying the publication, not as a full software package.
